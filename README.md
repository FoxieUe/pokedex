# Pokémon Pokédex



## Features



- Browse Pokémon by generations and type
- View detail of Pokémon information
- Display basic information and battle stats
- Previous / next Pokémon navigation



## Technologies used



- Python
- Flask
- Pandas
- Requests
- HTML
- CSS



## Future improvements

- Add wordle-style game
- Add more information in detail
